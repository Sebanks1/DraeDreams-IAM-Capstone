# DraeDreams IAM Capstone
### IAM · PAM · IGA — Microsoft Entra ID P2

A hands-on identity and access management portfolio demonstrating enterprise-grade controls across three disciplines, all implemented natively in Microsoft Entra ID P2 with no additional tools or licensing.

## What this covers

**IAM — Identity & Access Management**
- Identity provisioning from HR-driven attributes
- Joiner, mover, and leaver lifecycle automation
- Conditional Access enforcement across six policies
- Authentication method governance and SSPR

**PAM — Privileged Access Management**
- Break-glass emergency access with compensating controls
- Just-in-time role activation via Entra PIM
- Approval workflows, justification enforcement, and time-bound sessions

**IGA — Identity Governance & Administration**
- Entitlement management with access packages and quarterly reviews
- External identity governance for B2B guest accounts
- Identity risk protection with automated remediation policies
- Identity monitoring, audit logging, and 12-month log retention

## Platform
Microsoft Entra ID Premium P2 — single tenant, single platform
